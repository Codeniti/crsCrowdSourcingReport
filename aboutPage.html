What do we do?

Corporate investor require frequent updates on their investment. This applies to Social Causes as well, but majority of NGO's don't have expertise to create a system that could automate this task. This causes mistrust in investor and deter small investor. This can be resolved by creating a Web Based interface for investor and NGO which auto-generate daily/weekly/monthly/yearly report on development in project, based on updates posted by project manager.Such a kind of system would also create a marketplace for NGO projects.


How is it Beneficial?

  • This project will bring accountability into transaction of NGO which will provide positive incentive for investor
  • It will increase project visibility, since it aims to create a marketplace for NGO Project
  • Project will be stored as sequence of quantifiable tasks which can be easily tracked by Investor and Project Manager. This will bring in the rigour of Industrial Project that NGO most often lack
